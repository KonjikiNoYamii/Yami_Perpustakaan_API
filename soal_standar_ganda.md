# **✨ Soal & Jawaban Evaluasi Backend**

---

### **1. Apa peran utama Backend dalam sebuah aplikasi?**

a) Menampilkan antarmuka pengguna
b) Mengelola data, logika bisnis, dan komunikasi database
c) Menangani interaksi langsung dengan pengguna
d) Membuat animasi dan desain visual

**✅ Jawaban: b) Mengelola data, logika bisnis, dan komunikasi database**

---

### **2. HTTP Method POST digunakan untuk operasi apa dalam CRUD?**

a) Read
b) Update
c) Create
d) Delete

**✅ Jawaban: c) Create**

---

### **3. Apa arti dari HTTP Status Code 201 Created?**

a) Permintaan berhasil diproses
b) Data baru berhasil dibuat
c) Data tidak ditemukan
d) Terjadi kesalahan server

**✅ Jawaban: b) Data baru berhasil dibuat**

---

### **4. Apa itu Node.js?**

a) Bahasa pemrograman frontend
b) Framework JavaScript untuk database
c) Runtime environment untuk menjalankan JavaScript di luar browser
d) Library untuk membuat antarmuka pengguna

**✅ Jawaban: c) Runtime environment untuk menjalankan JavaScript di luar browser**

---

### **5. Engine JavaScript apa yang digunakan oleh Node.js?**

a) SpiderMonkey
b) Chakra
c) V8
d) JavaScriptCore

**✅ Jawaban: c) V8**

---

### **6. Middleware express.json() di Express.js digunakan untuk apa?**

a) Mengamankan header HTTP
b) Memparsing body request dalam format JSON
c) Melakukan logging HTTP request
d) Menangani error global

**✅ Jawaban: b) Memparsing body request dalam format JSON**

---

### **7. Perbedaan utama antara Route Params (/users/:id) dan Query String (/users?name=...) adalah…**

a) Route Params untuk filter, Query String untuk identifikasi resource
b) Route Params untuk identifikasi resource, Query String untuk filter/sorting
c) Keduanya sama-sama untuk identifikasi resource
d) Keduanya sama-sama untuk filter data

**✅ Jawaban: b) Route Params untuk identifikasi resource, Query String untuk filter/sorting**

---

### **8. Apa fungsi utama dari Global Error Handler di Express.js?**

a) Memvalidasi input dari pengguna
b) Mencatat semua request HTTP yang masuk
c) Menangkap semua error yang terjadi di aplikasi dan mengirim respons yang konsisten
d) Mengamankan aplikasi dari serangan XSS

**✅ Jawaban: c) Menangkap semua error yang terjadi di aplikasi dan mengirim respons yang konsisten**

---

### **9. Manfaat utama pemisahan Controller, Service, dan Route dalam arsitektur MVC + Service Layer adalah…**

a) Membuat kode lebih pendek
b) Meningkatkan keterbacaan, pemeliharaan, dan skalabilitas kode
c) Mengurangi jumlah file dalam project
d) Hanya berfungsi untuk aplikasi kecil

**✅ Jawaban: b) Meningkatkan keterbacaan, pemeliharaan, dan skalabilitas kode**

---

### **10. Apa tanggung jawab utama dari Controller dalam arsitektur MVC?**

a) Menjalankan logika bisnis dan berinteraksi dengan database
b) Mendefinisikan struktur data aplikasi
c) Menerima request HTTP, mengambil data, dan menyuruh Service untuk logika bisnis, lalu mengirim respons
d) Menangani validasi input secara langsung

**✅ Jawaban: c) Menerima request HTTP, mengambil data, dan menyuruh Service untuk logika bisnis, lalu mengirim respons**
